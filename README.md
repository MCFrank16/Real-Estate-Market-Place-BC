## Steps to get up and run the project
```
    > clone the repository: https://github.com/MCFrank16/Real-Estate-Market-Place-BC.git
    > install dependencies with: npm install
    > cd eth-contracts
    > make sure the ganache is up and running on port 7545 or change the truffle-config.js to your corresponding port
    > Run Command: truffle compile
    > For testing: truffle test
```
